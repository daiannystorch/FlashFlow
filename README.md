## 🔀 Evolução do Projeto: Branch de Idiomas

Além deste projeto principal voltado para a avaliação da pós-graduação, a estrutura deste código foi reaproveitada e expandida para um caso de uso real e pessoal. 

Foi criada uma branch paralela chamada `Hiragana`, onde a lógica de JavaScript foi adaptada para criar um baralho completo de estudos do alfabeto japonês básico (Gojūon), com 46 cartas e adaptações de CSS para melhor legibilidade de caracteres asiáticos.
