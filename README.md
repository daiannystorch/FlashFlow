# FlashFlow 🧠⚡️

Um aplicativo web interativo de flashcards desenvolvido para otimizar o estudo e a memorização de conceitos de programação. Projeto prático desenvolvido como parte de avaliação de pós-graduação.

## 💻 Sobre o Projeto

O FlashFlow permite que os usuários testem seus conhecimentos através de cartões interativos. O projeto foi construído traduzindo fielmente um design do Figma para código, aplicando conceitos modernos de interface de usuário (UI) e interatividade com JavaScript.

### ✨ Funcionalidades
* **Interatividade Front/Back:** Clique no cartão para revelar a resposta com uma transição suave de cores e layout.
* **Navegação Dinâmica:** Botão para avançar pelos flashcards, atualizando os dados na tela instantaneamente.
* **Contador Automático:** O sistema rastreia e exibe a posição atual do usuário no baralho (ex: "Card 1 de 5").
* **Design Fiel:** Estilização baseada em protótipo do Figma, utilizando variáveis CSS, Flexbox e web fonts (Google Fonts).

## 🛠 Tecnologias Utilizadas

Este projeto foi construído do zero utilizando as tecnologias fundamentais da web:

* **HTML5:** Estruturação semântica do conteúdo e divisão lógica em caixas (DOM).
* **CSS3:** Estilização completa, variáveis de cor (`:root`), Flexbox para alinhamento centralizado e transições (`transition`) para efeitos de hover e virada do cartão.
* **JavaScript:** Lógica de programação para manipulação do DOM, escuta de eventos (`addEventListener`), alternância de classes CSS (`classList.toggle`) e renderização dinâmica de conteúdo a partir de uma Array de objetos.

*Desenvolvido com dedicação para consolidação de conhecimentos em Front-end.*
